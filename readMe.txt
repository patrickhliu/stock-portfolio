Project 1 from Harvard CS75
http://cdn.cs75.net/2012/summer/projects/1/

Project Details:
Create a website where users can register an account to...
1) look up stock quotes
2) purchase stocks
3) sell stocks
4) view the net worth of their portfolio

All new users start with $10,000 in their account balance.

Languages: HTML / CSS / jQuery / PHP / SQL
3rd party: yahoo stock lookup

"finance.sql" contains the database w/ 3 tables: 
1) portfolio table - store information on a user's portfolio
2) stocks - store information on all stocks that are looked up
3) users - store information for each account holder
